## 命令行



* 常用命令行及作用

  1.ls :列出当前目录下的文件或文件夹。-l显示详细信息，-a显示所有文件包括隐藏文件

  2.cd:进入目录；cd..返回上一级

  3.pwd：显示当前工作目录的路径

  4.mkdir：创建新目录

  5.rm：删除文件或目录；可以使用-i在删除前确认

  6.cp：复制文件或目录；可以使用-r选项递归复制目录

  7.kill:结束进程

  8.clear；清空终端屏幕

  9.find:寻找文件
