# id

id在css中是唯一的；

id选择器可以为标有特定ID的html元素指定特定的样式。

css中id选择器以“#”来定义

例如：

~~~<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8"> 
<title>菜鸟教程(runoob.com)</title> 
<style>
#para1
{
	text-align:center;
	color:red;
} 
</style>
</head>

<body>
<p id="para1">Hello World!</p>
<p>这个段落不受该样式的影响。</p>
</body>
</html>
~~~

在这串代码中使用了#来定义了html元素的样式

通过使用来改变html的表现效果
